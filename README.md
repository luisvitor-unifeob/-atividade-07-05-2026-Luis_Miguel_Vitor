## Sobre o Projeto

Sistema de informação para gestão de uma rede de cinemas, contemplando controle de filmes, sessões e registro de público.

## Como Executar

```bash
cd rede_cinemas/src
python main.py
```

Requer Python 3.8+. Sem dependências externas, utiliza SQLite nativo do Python.

## Funcionalidades

- Cadastrar cinemas, filmes e sessões
- Registrar público em sessões
- Listar filmes em cartaz por cinema
- Consultar total de público por sessão e por cinema
- Validação de conflito de horário entre sessões
- Validação de capacidade máxima do cinema
