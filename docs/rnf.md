### RNF01 — Persistência de Dados
O sistema deve persistir todos os dados em banco de dados SQLite, garantindo integridade entre as entidades.

### RNF02 — Arquitetura em Camadas
A arquitetura do sistema deve seguir o padrão MVC com camadas de Service e Repository bem definidas.

### RNF03 — Interface de Linha de Comando
O sistema deve ser executável via linha de comando, sem necessidade de interface gráfica.

### RNF04 — Organização Modular
O código deve ser organizado em módulos separados por responsabilidade, facilitando manutenção e evolução.
